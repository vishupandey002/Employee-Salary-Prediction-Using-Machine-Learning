# Employee Salary Prediction Using Machine Learning

## 📌 Project Overview

This project predicts employee salaries based on various factors such as job title, years of experience, education level, skills, industry, company size, work location, remote work status, and certifications using **Multiple Linear Regression**.

The project demonstrates a complete Machine Learning workflow, including data preprocessing, exploratory data analysis (EDA), feature engineering, model training, evaluation, and prediction.

---

## 📂 Dataset Features

- Job Title
- Experience Years
- Education Level
- Skills Count
- Industry
- Company Size
- Location
- Remote Work
- Certifications
- Salary (Target Variable)
